Comment Numerical Captcha Plugins

== Description ==

Adds Numerical addition CAPTCHA anti-spam methods to WordPress on the comment form.
In order to post comments, users will have to type in the phrase shown on the addition.
This can help prevent spam from automated bots.

[Plugin URI]: (http://www.atgleam.com)

V1.1
   * fix : Modified interface pos.

== Installation ==

1. Upload the `wp-num-captcha` folder to the `/wp-content/plugins/` directory

2. Activate the plugin through the `Plugins` menu in WordPress

3. If you ever have to upgrade, simply deactivate, uninstall, and repeat the installation steps with the new version. Or you can click on "Upgrade Automatically" if prompted from the admin menu.
